## Log

- [x] Display task
  ```
  scd log task_name 
  ```
- [x] Display category
  ```
  scd log @category
  ```
- [x] Display all tasks
  ```
  scd log -a
  ```
- [ ] Display filtered tasks **Deadline**
  ```
  scd log -f deadline 20-04-2069
  ```
- [ ] Display filtered tasks **Status**
  ```
  scd log -f status Ongoing
  ```
