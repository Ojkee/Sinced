## Log

- [x] Display task
  ```
  scd log task_name 
  ```
- [x] Display category
  ```
  scd log @category
  ```
- [x] Display all tasks
  ```
  scd log -a
  ```
- [x] Display filtered tasks **Deadline**
  ```
  scd log -deadline 20-04-2069
  ```
- [x] Display filtered tasks **Status**
  ```
  scd log -status Ongoing
  ```
